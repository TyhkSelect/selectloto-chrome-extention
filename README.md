# LOTO6 購入入力補助 Chrome拡張

selectloto.jp の「LOTO6 抽せん回ごとのセット一覧」ページから、組み合わせ数字と口数を読み取り、宝くじ公式サイト（みずほ銀行）の購入入力フォームへ自動入力する Chrome 拡張機能です。

## 機能

- セット一覧ページの組み合わせを一覧表示・選択
- 公式サイトの数字ボタンを自動クリックして入力
- 口数を自動設定
- 25組を超える場合はカート追加 → 「買い物を続ける」を自動処理して続きを入力

## インストール方法

### 1. ファイルをダウンロード

- このページ右上の **Code → Download ZIP** でダウンロード
- ZIPを解凍して任意の場所に保存

### 2. Chrome に読み込む

1. Chrome のアドレスバーに `chrome://extensions/` と入力して開く
2. 右上の **「デベロッパーモード」をON**（トグルをクリック）
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. 解凍したフォルダ（`loto6-chrome-extension`）を選択

アドレスバー右側に拡張アイコンが表示されれば完了です。

> **Windows でも同じ手順で使えます。**

## 使い方

1. selectloto.jp の **「LOTO6 抽せん回ごとのセット一覧」** ページを開く
2. データが読み込まれたら、Chrome の拡張アイコンをクリック
3. 組み合わせ一覧が表示される → 購入したいものにチェック
4. **「公式サイトで入力開始」** ボタンをクリック
5. 宝くじ公式サイトの **LOTO6購入入力ページ** を開く
6. 自動で数字と口数が入力される
7. 内容を確認して **「お支払い内容のご確認へ」** をクリック

### 25組を超える場合

- 25組入力後に自動で **「カートに入れる」** をクリック
- 確認ページで自動で **「買い物を続ける」** をクリック
- LOTO6購入ページへ戻ると残りを自動入力
- 全組み合わせ入力完了後にお支払いへ進んでください

## 注意事項

- 最終的な購入確定（お支払い）はご自身でお願いします
- 口数が10を超える場合は10口に丸めます（公式サイトの上限）
- Chrome のデベロッパーモード警告が表示される場合がありますが動作に影響はありません
- 本拡張は個人利用を目的としています。公式サイトの利用規約をご確認の上ご使用ください

## 動作環境

- Google Chrome（Mac / Windows）
- selectloto.jp アカウントが必要
- 宝くじ公式サイト（みずほ銀行）のアカウントが必要
